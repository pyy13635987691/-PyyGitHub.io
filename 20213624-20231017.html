再linux中的文件路径与文件名中的空格要用到的转义字符来进行转义
    Linux和Windows的存储路径表示方式确实有一些区别。下面是它们的主要区别：

1. 路径分隔符：
   - Linux使用正斜杠（/）作为路径分隔符，例如：/home/user/documents。
   - Windows使用反斜杠（\）作为路径分隔符，例如：C:\Users\Documents。

2. 根目录表示：
   - Linux的根目录用单斜杠（/）表示，即所有目录的起始点。
   - Windows的根目录使用盘符加反斜杠（\）表示，例如：C:\。

3. 相对路径和绝对路径表示：
   - Linux中，相对路径是相对于当前工作目录的路径，以当前目录（.）或上级目录（..）作为起点。
        例如，如果当前目录是/home/user，那么documents目录可以表示为./documents。
   - Windows中，相对路径也是相对于当前工作目录的路径，但是使用反斜杠作为分隔符。
        例如，如果当前目录是C:\Users，那么Documents目录可以表示为.\Documents。

4. 特殊目录表示：
   - 在Linux中，有一些特殊的目录表示方式，例如家目录（~）表示当前用户的主目录，
        根目录（/）表示文件系统的起始点。
   - 在Windows中，有一些特殊的目录表示方式，例如用户文件夹（%USERPROFILE%）表示当前用户的主目录，
        根目录（C:\）表示文件系统的起始点。

    需要注意的是，虽然Linux和Windows在存储路径上有一些区别，但大多数现代编程语言和工具都提供了跨平台的函数和库来处理路径，
并且可以自动适应

二、知识点二

                
    JavaScript中的函数是一段可重复使用的代码块，具有特定的功能。函数可以接收参数、执行一系列的操作，并返回一个结果。

以下是JavaScript中函数的常见用法和功能：

1. 定义函数：使用关键字 function 来定义一个函数，可以指定函数名、参数列表和函数体。

2. 调用函数：通过函数名加上括号 () 来调用函数，并可以传递参数供函数使用。

3. 参数传递：函数可以在定义时指定参数列表，当函数被调用时，可以传递实际的参数值给形式参数，供函数内部使用。

4. 返回值：函数可以使用关键字 return 来返回一个值，返回值可以在函数调用处被捕获和使用。

5. 匿名函数：可以创建没有函数名的匿名函数，通常作为另一个函数的参数或赋值给变量使用。

6. 高阶函数：JavaScript中函数也是一种数据类型，可以将函数作为参数传递给其他函数，或者将函数作为返回值返回。

7. 闭包：函数内部可以访问外部函数的变量和参数，形成闭包。这使得函数可以访问其创建时的环境变量，即使外部函数已经执行完毕。

8. 函数方法：函数也是对象，因此可以具有自己的属性和方法。例如，可以使用 Function.prototype 对函数进行扩展和修改。

9. 递归：函数可以直接或间接地调用自身，这称为递归。递归函数可以解决一些需要重复操作的问题。

10. 回调函数：在异步操作中，可以将一个函数作为参数传递给另一个函数，并在适当的时候调用该函数，这被称为回调函数。

    函数在JavaScript中具有非常重要的地位，可以用于模块化代码、封装功能、提高代码复用性和可维护性等。通过合理使用函数，可
以使代码更加清晰、简洁和易于理解。

三、document与element的简介

                
    在JavaScript中， document 是一个表示当前HTML文档的对象，可以通过它来直接操作HTML文档中的元素、样式、内容等。
使用  document  对象，可以实现一些常见的操作，例如查找HTML元素、修改页面内容、修改样式等。
    以下是对  document  对象一些常用的属性和方法进行介绍：

1. document.getElementById(id) ：根据id查找元素，返回一个对象。
    例如：  var element = document.getElementById("myId") 。

2. document.getElementsByClassName(className) ：根据类名查找元素，返回一个 HTMLCollection 对象，其中包含所有匹配的元素。
    例如： var elements = document.getElementsByClassName("myClass") 。

3. document.getElementsByTagName(tagName) ：根据标签名查找元素，返回一个 HTMLCollection 对象，其中包含所有匹配的元素。
    例如： var elements = document.getElementsByTagName("div") 。

4. document.getElementsByName(name) ：根据name属性查找元素，返回一个 NodeList 对象，其中包含所有匹配的元素。
    例如： var elements = document.getElementsByName("myName") 。

5. document.querySelector(selector) ：根据 CSS 选择器查找第一个匹配的元素，返回一个对象。
    例如： var element = document.querySelector(".myClass#myId") 。

6. document.querySelectorAll(selector) ：根据 CSS 选择器查找所有匹配元素，返回一个 NodeList 对象，其中包含所有匹配元素。
    例如： var elements = document.querySelectorAll(".myClass") 。

7. document.createElement(tagName) ：创建一个新的HTML元素，返回一个新的对象。
    例如： var element = document.createElement("div") 。

8. element.parentNode ：获取元素的父节点，返回一个对象。
    例如： var parent = element.parentNode 。

9. element.childNodes ：获取元素的所有子节点，返回一个 NodeList 对象。
    例如： var children = element.childNodes 。

10. element.setAttribute(name, value) ：设置元素的属性值。
    例如： element.setAttribute("class", "myClass") 。

11. element.style.property = value ：设置元素的 CSS 样式属性值。
    例如： element.style.color = "#000000" 。

12. element.innerHTML ：获取或设置元素的 HTML 内容。
    例如： var htmlContent = element.innerHTML 、 element.innerHTML = "<p>与</p>>" 。

13. element.innerText ：获取或设置元素的纯文本内容。
    例如： var textContent = element.innerText 、 element.innerText = "Hello, world!" 。

14. element.textContent ：获取或设置元素的文本内容，功能与  innerText  类似。
    例如： var textContent = element.textContent 、 element.textContent = "Hello, world!" 。

    以上是一些常用的  document  对象的属性和方法，可以帮助开发者更加便捷地操作HTML文档，实现各种自定义的页面交互效
果。
